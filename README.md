### Quick Linux Setup Steps:
```
- apt install npm
- apt  install gh
- git clone https://github.com/CryptoGnome/Bybit-Lick-Hunter-v4.git
- cd Bybit-Lick-Hunter-v4
- npm install
- cp example.env .env
- edit settings
- node app.js
```
